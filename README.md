## Customization

### Colors
The professional color palette can be customized by modifying CSS custom properties in `styles.css`:

```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #34495e;
    --accent-color: #3498db;
    --accent-hover: #2980b9;
    --text-primary: #333;
    --text-secondary: #555;
    --background-primary: #fff;
    --background-secondary: #f8f9fa;
}
```

Update the content in `index.html` to reflect your own professional information:
- Personal details and professional bio

## Deployment
This is a static portfolio site that can be deployed to:
- GitHub Pages (ideal for personal portfolios)
- Netlify
- Vercel
- Any static hosting service

For GitHub Pages deployment:
1. Push the code to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Select the main branch as the source
4. Your portfolio will be available at `https://yourusername.github.io/repository-name`
